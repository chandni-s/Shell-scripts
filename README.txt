# Guessing Game:
# It asks the user for input upto 6 times and tries to match
# with the number that computer chooses randomly. If integer
# chosen by user is too small or big, it output 'Too small'
# or 'Too big' respectively, and allows user to re-enter.
# If random number chosen by computer matches user-input, it
# output 'You Win' and exits the game

# To run this script:
# ./guessing_game.sh
